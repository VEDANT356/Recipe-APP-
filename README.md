🍲 Recipe App

A simple and responsive Recipe Web App built using HTML, CSS, and JavaScript.
Users can search for recipes and view details easily.

🚀 Features

🔍 Search recipes by name

📖 View recipe details

🎨 Clean and responsive UI

⚡ Fast and lightweight

🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

API Integration



📂 Project Structure
Recipe-App/
│── index.html
│── style.css
│── index.js
│── README.md


🧠 How It Works

User enters a recipe name in the search box

App fetches data from API

Recipe details are displayed on the screen
